# Stratum 架构偏移日志（DRIFT-LOG）

> **用途：** 记录所有检测到的规范冲突、架构偏移、临时豁免和实际实现与规范不一致的情况。
> **记录时机：** AI 或人工发现规范冲突时，立即在此记录后等待人工决策。
> **处置流程：** 见 FORBIDDEN.md 第 5 节。

---

## 日志格式说明

```markdown
### DRIFT-XXX：标题（日期）

- **发现方式**：AI 检测 / 人工审查
- **发现节点**：NODE-XXX 或会话日期
- **违规类型**：规范冲突 / 实现偏移 / 命名偏差 / 依赖方向错误
- **描述**：<具体描述>
- **涉及文件**：
- **关联规范**：SPEC-XXX / F-XXX
- **处置结果**：待定 / 已修正 / 豁免（见 ADR-XXX）
- **处置日期**：
```
