## ADR 编写模板

新增决策时，在 `adr/` 目录下按 `ADR-XXX-标题.md` 格式创建文件，内容遵循以下模板：

```markdown
# ADR-XXX：决策标题

## 状态
[提案中 / 已接受 / 已废弃 / 已替代]

## 背景
<为什么需要做这个决策，面临什么问题>

## 决策
<最终选择是什么>

## 替代方案（已考虑）
- 方案 A：优缺点
- 方案 B：优缺点

## 权衡
<为什么选择当前决策而非替代方案>

## 影响
<对架构、规范、开发流程的影响>

## 豁免说明（如适用）
<若本 ADR 是对 FORBIDDEN.md 某条的豁免，说明豁免范围和条件>

## 关联规范
<涉及的 specId 或文档>

_记录日期：YYYY-MM-DD | 记录人：_
```
